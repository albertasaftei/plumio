Version 2.2.0 of plumio is here!

<!-- truncate -->

This release includes:

### Major highlights

- Light/dark mode switcher. Inside Settings -> Account
- Resizable sidebar in case nested folders get messy
- Dedicated section for build info with commit, commit date, commit branch inside Settings -> Build information. To help the user know which version is currently using
- Reworked Popover UI to result more compact and easier to use. Moved create file/folder buttons inside the popover as well
- Checkboxes text now have strikethrough style and less opacity if the state is "checked"
- Editor Toolbar -> some buttons were adding but not removing stylings
- Sidebar -> improved the UI with a more compact version

---

### Security

- Fixed an inconsistency bug on the JWT management flow. On logout it wasn't being removed from the local storage, thus logging in with another account resulted in having data from the previous one
