Version 2.3.0 of plumio is here!

<!-- truncate -->

This release includes:

### Major highlights

- Reconciliation process to automatically fix DB records that point to missing files, and remove records for deleted documents. This process runs daily and can also be triggered manually from the admin panel.
- Added Full-text search section to search for keywords across all documents, with results showing document title and a content snippet.
- Refactored sidebar for a more streamlined design and better organization of features.
