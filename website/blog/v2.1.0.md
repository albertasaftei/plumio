Version 2.1.0 of plumio is here!

<!-- truncate -->

### New Editor Features:

- Toolbar with clickable buttons for common markdown formatting options
- Text highlighting
- Links
- Checkboxes
- Horizontal rules
- Codeblocks now have copy button and language selector

### Demo UI changes:

Removed the top banner and replaced it with a more compact info box on the bottom right corner.

### Other improvements:

Minor bug fixes and performance improvements.
