Version 2.5.0 of plumio is here!

{/* truncate */}

This release includes:

### Major highlights

- Added **Mermaid diagrams support**. Add Mermaid code in the Plain view using a ` ```mermaid ` code block and see the live preview in the Live view.
- Added the ability to **move documents and folders** in the sidebar by opening the item menu and selecting "Move". This opens a dialog where you can select the destination folder.
- Added the ability to **attach files** to documents.
  - PDFs can be rendered inline with a preview and pagination controls.
  - Images can be rendered inline as well.
  - Other file types can be downloaded.

### Minor improvements

- Restyled 404 page to match app design.

### Bugs

- Restoring deleted documents wasn't restoring the metadata file, thus causing the document to be restored without colors and favorite tag.
- Fixed a bug where there was no way to disable highlight and text color modes unless breaking line and returning on the previous one.
- Fixed math equations not rendering correctly.

### Security

- Updated dependencies to fix security vulnerabilities.

Thank you for using plumio! 🎉
